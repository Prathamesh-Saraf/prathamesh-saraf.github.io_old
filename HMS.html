A Wearable Health Monitoring Device

A wearable device to obtain the heart rate, blood pressure, body temperature, oxygen level, and GPS location of the person. The data will be transmitted to a 2-3km far local base station.

  • Integrated and programmed the respective sensors using Arduino pro mini onto a wrist watch sized wearable device and displayed the values on a TFT LCD screen attached to the device.
  • Transmitted the data from the wearable device to a manpack module (Raspberry Pi) using Bluetooth.
  • Transmitted the data from the manpack module to a local base station using the LoRa module.
