    Dynamic Optimal Control of a Quadrotor

    The project focuses on stability analysis of a Quadrotor for varied flying conditions.

  • Formulated the Quadrotor dynamics and modelled the mathematical state space equations.
  • Designed, tuned and simulated the PID controller & LQR optimal for the Quadrotor.
  • Validated the controller responses for various flying and disturbance scenarios and presented comparative results for the two controllers.

    A CNN Approach Towards Self-driving Cars

    The project aims at building a level-2 autonomous vehicle which would take the lane markings from an on-board camera and the nearby obstacles from 3 Ultrasonic sensors as input. The input image is processed and a steering angle is obtained.

  • Designed and assembled the complete hardware setup.
  • Programmed the obstacle detection and avoidance algorithm using the steering angle and the ultrasonic sensor input.
  • Programmed the vehicular motion for different driving scenarios like lane change, traffic adaption, curved path motion, etc.

    A Wearable Health Monitoring Device

    A wearable device to obtain the heart rate, blood pressure, body temperature, oxygen level, and GPS location of the person. The data will be transmitted to a 2-3km far local base station.

  • Integrated and programmed the respective sensors using Arduino pro mini onto a wrist watch sized wearable device and displayed the values on a TFT LCD screen attached to the device.
  • Transmitted the data from the wearable device to a manpack module (Raspberry Pi) using Bluetooth.
  • Transmitted the data from the manpack module to a local base station using the LoRa module.

    Electronics Subsystem Design, Hyperloop India

    [Certificate]

    The work aims at designing and building a pod prototype for the Hyperloop SpaceX competition. The Hyperloop India team consists of Mechanical, CFD, Propulsion, Software and Electronics Subsystems. As a member of Electronics subsystems, I:

  • Worked on sensor integration for navigation, telemetry, and communication system used in the prototype.
  • Studied and designed the Linear Induction Motors for the propulsion of the pod.
  • Worked on Battery Management System consisting of power distribution, fault detection and prevention for the pod.
  • Built the Active suspension (stability) system using PID controllers for the pod.
